## About

This repository contains my Power BI projects created during my training at the Data Technician Bootcamp. These reports focus on business intelligence, data modelling, and building interactive dashboards that support data-driven decision-making.

### Built With

[![Power BI](https://img.icons8.com/?size=100&id=Ny0t2MYrJ70p&format=png&color=000000)](https://powerbi.microsoft.com/)

### üß† Skills Demonstrated (Power BI Focus)

During my training, I developed practical Business Intelligence skills using **Power BI Desktop**. This includes **data modelling** by creating relationships between multiple tables, building **DAX measures** for calculated fields and KPIs, and designing effective visuals such as **line charts, stacked bar charts, treemaps, and map visuals**. I also focused on **dashboard interactivity and consistent formatting** to create clear, user-friendly reports that support strong data storytelling.
